# Sobre o projeto

**Invoice Unlocker** é uma plataforma inovadora para envio de propostas comerciais no formato de páginas web interativas, protegidas por códigos de acesso simplificados.

O objetivo é substituir o envio de arquivos estáticos (PDF/PPT) por uma experiência digital imersiva, moderna e segura, eliminando a fricção de sistemas de login complexos e entregando uma primeira impressão de alto impacto ("Wow Effect") com design em glassmorphism e micro-interações.
